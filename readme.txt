tutorial menggunakan script:
1. buka termux
2. install modul dg cara ketik dibawan ini:
   pip install telethon
   pip install requests
   pip install random
   pip install colorama   
3. buka script yg ada di penyimpanan device dg cara ketik dibawah ini:
   cd /sdcard/namafolderpenyimpananscript
   ls
4. kemudian jalankan script dg cara ketik dibawah ini
   python create.py
